
# Hoshimi Bot

[![Built with hoshimi](https://img.shields.io/badge/built%20with-hoshimi-8B5CF6?style=for-the-badge&logo=npm&logoColor=white)](https://www.npmjs.com/package/hoshimi)
[![Discord Bot](https://img.shields.io/badge/type-discord%20bot-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.com/)

This project is a simple example Discord bot built to test and demonstrate features of the `hoshimi` package.

Package URL: https://www.npmjs.com/package/hoshimi

## Purpose

The bot is mainly used as a playground for testing features, commands, and integrations around `hoshimi`.

## Main Stack

- `hoshimi` for the bot framework and core features
- `seyfert` for command and bot structure
- `redis` for shared storage and state
- `dotenv` for environment variables

## Available Scripts

- `pnpm start` - run the bot
- `pnpm dev` - run the bot in debug mode
- `pnpm typecheck` - check TypeScript types
- `pnpm lint` - lint and write fixes with Biome
- `pnpm format` - format the project with Biome

## Notes

This repository is intentionally small and experimental, so the code may change often while new ideas are tested.
